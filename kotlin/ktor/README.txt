This is for HTTP server environment. It shows how to do HTTP
listening, regexp and testing in Kotlin with ktor.

Getting started:
1. cd to this directory
2. `./gradlew build` - build project
3. Import `build.gradle` in your favourite IDE
4. Run StartupWebServerTest as a test in your IDE
5. Run StartupWebServer as a Java main class
6. Go and see the server in action:
   http://localhost:8080/?q=what+is+the+sum+of+8+and+22
7. Try to modify the number in the above URL and verify
   that the answer is correct
